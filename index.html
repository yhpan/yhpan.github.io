新建博客

Github Bash 定位到我们存放博客代码等文件夹根目录（如Blogs），执行命令：

hexo new ‘my-first-blog’

hexo会帮我们在_posts下生成相关md文件，我们只需要打开这个文件就可以开始写博客了，可以修改最开始的front-matter内容：
